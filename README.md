# بک‌اند فروشگاه Jordan Gallery Shop

این سرور دو کار را انجام می‌دهد:
1. ثبت‌نام و ورود کاربران (رمز عبور به‌صورت هش‌شده با bcrypt ذخیره می‌شود، نه متن ساده)
2. اتصال به درگاه پرداخت زرین‌پال (ایجاد تراکنش و تایید پرداخت)

## راه‌اندازی محلی (تست روی سیستم خودت)

```bash
cd backend
npm install
cp .env.example .env
# مقادیر داخل .env را با اطلاعات خودت پر کن
npm run dev
```

سرور روی `http://localhost:4000` بالا می‌آید.

## گرفتن مرچنت‌آیدی زرین‌پال

1. در [zarinpal.com](https://www.zarinpal.com) ثبت‌نام کن و پذیرنده بساز.
2. برای تست، از حالت «آزمایشی / sandbox» زرین‌پال استفاده کن تا نیازی به تایید نهایی نباشد.
3. مرچنت‌آیدی را در `ZARINPAL_MERCHANT_ID` قرار بده.

## دیپلوی رایگان (Render)

1. یک ریپازیتوری گیت‌هاب بساز و پوشه‌ی `backend` را در آن push کن.
2. در [render.com](https://render.com) یک "Web Service" جدید بساز و ریپازیتوری را وصل کن.
3. تنظیمات:
   - Build Command: `npm install`
   - Start Command: `npm start`
4. متغیرهای محیطی (`JWT_SECRET`, `ZARINPAL_MERCHANT_ID`, `CALLBACK_URL`, `FRONTEND_URL`) را در بخش Environment پنل Render وارد کن.
5. پس از دیپلوی، آدرس سرور چیزی شبیه `https://jordan-gallery-shop-backend.onrender.com` خواهد بود.

⚠️ **نکته مهم درباره‌ی پلن رایگان Render:** فضای دیسک در پلن رایگان موقتی است و با هر ری‌استارت سرور، فایل `store.db` (و اطلاعات کاربران/سفارش‌ها) ممکن است پاک شود. برای فروشگاه واقعی که داده‌ها باید دائمی بمانند، پیشنهاد می‌شود از یک دیتابیس ابری رایگان مثل **Supabase** یا **MongoDB Atlas** استفاده کنی. اگر خواستی، می‌توانم همین بک‌اند را با Supabase به‌جای SQLite بازنویسی کنم.

## اتصال به فرانت‌اند

در فایل فروشگاه، مقدار `API_BASE_URL` را برابر آدرس همین بک‌اند (بعد از دیپلوی) قرار بده.

## چک‌لیست امنیتی پیش از رفتن به حالت واقعی (Production)

- [ ] `JWT_SECRET` را حتماً عوض کن (مقدار پیش‌فرض را نگه نداری)
- [ ] از حالت آزمایشی زرین‌پال خارج شو و مرچنت‌آیدی واقعی را جایگزین کن
- [ ] HTTPS روی دامنه‌ات فعال باشد (اکثر سرویس‌های هاست رایگان به‌صورت خودکار این را فراهم می‌کنند)
- [ ] یک دیتابیس دائمی (نه SQLite روی دیسک موقت) برای نگهداری کاربران و سفارش‌ها استفاده کن
